# MediAssist IA

Projeto acadêmico desenvolvido para a disciplina de Desenvolvimento de Software.

## Objetivo

Realizar integração com a API Gemini para responder dúvidas sobre medicamentos.

## Tecnologias

- Python
- Gemini API
- Pytest
- GitHub Actions

## Execução

Instalar dependências:

pip install -r requirements.txt

Executar:

python app/main.py
